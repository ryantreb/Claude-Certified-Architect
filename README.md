# Dragon Age: Legends — an Architect's Study Saga

A single-file, dark-fantasy browser RPG (codename *Weavefall*) that retells the official
Dragon Age Legends campaign — defending Kaiten for Viscount Ravi against a demon of Pride —
while drilling two agentic-AI certifications through turn-based combat, an explorable
isometric overworld, and a keep you rebuild:

- **Claude Certified Architect — Foundations** (pass 720/1000) — *The Architect's Vigil*
- **GitHub Certified: Agentic AI Developer, GH-600 beta** (pass 700/1000) — *The Marcher Campaign*

Every reward is tied to real recall: concepts are tracked with per-track SM-2-style spaced
repetition, mastery requires 3+ correct recalls on separate *due* reviews, cramming earns
nothing, and the two pass meters are computed purely from earned mastery weighted by each
exam's domain weights.

## Play

Open **`index.html`** in any modern browser — that's it. No build, no install, no network.
Works on phone (portrait, touch) and desktop (mouse/keyboard, WASD/arrows). Saving uses
`localStorage` when available and degrades gracefully when it isn't.

An original fan-made study game. Not affiliated with or endorsed by any vendor.
